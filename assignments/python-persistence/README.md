# 📘 Assignment: Python Data Persistence

## 🎯 Objective

Learn how to store and manage data in a SQLite database using Python so programs can keep information between runs.

## 📝 Tasks

### 🛠️ Create a database and contact table

#### Description
Create a SQLite database file and define a table for storing contact information.

#### Requirements
Completed program should:

- Create or open a database file named `contacts.db`
- Create a `contacts` table with columns for `id`, `name`, `email`, and `phone`
- Confirm the table was created successfully before adding records


### 🛠️ Add and view contacts

#### Description
Insert sample contact records into the database and display them.

#### Requirements
Completed program should:

- Insert at least three contact records into the `contacts` table
- Query all contacts from the database
- Print each contact in a readable format


### 🛠️ Update and delete records

#### Description
Modify a contact's information and remove one contact from the database.

#### Requirements
Completed program should:

- Update a contact record by `id`
- Delete one contact record by `id`
- Display database contents after each change
